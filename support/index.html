Holiday Expense Manager 2.0 Support

Support page draft for App Store submission.

iCloud

To use iCloud sync and sharing, sign in to iCloud on your device and allow iCloud Drive/CloudKit services for the app.

Add a trip

Open the app, choose Create Trip, enter the trip name, dates, base currency and optional budget.

Add an expense

Use Add Expense from Home or from a trip. Select a trip if needed, enter amount, currency, date and optional details.

Receipt OCR

You can attach a receipt image or PDF and run local OCR. OCR suggestions can be reviewed before applying them to the expense form.

Sharing

Trip sharing uses Apple's iCloud Sharing. Use Share with iCloud from the trip Sharing screen. A .hemshare file is only a preview and does not grant access.

Diagnostics logs

Open Settings → Diagnostics → App Logs to copy local diagnostic logs for support.
