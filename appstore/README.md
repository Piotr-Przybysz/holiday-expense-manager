# App Store Submission Materials

This folder contains App Store Connect / App Review preparation materials for Holiday Expense Manager 2.0 Build 31.

## Files

- `AppReviewNotes.md` — notes to paste into App Review.
- `AppPrivacy.md` — proposed App Privacy answers for manual confirmation.
- `Metadata.md` — App Store metadata draft.
- `ScreenshotsChecklist.md` — screenshots checklist.
- `ExportCompliance.md` — export compliance recommendation.
- `ReleaseChecklist.md` — release readiness checklist.
- `CloudKitReviewChecklist.md` — CloudKit/iCloud review checklist.
